1.2.1.3 Indexación de vectores

Cómo extraer elementos de un vector - R es un lenguaje de programación indexado en 1; es decir, al primer elemento de un vector se le asigna la posición 1. Para extraer elementos de un vector usamos corchetes []

Gráfico de indexación de vectores

1.2.1.3.1 Por posición
Nombre_vector[3]
Nombre_vector[c(3)]
Nombre_vector[1:4]
Nombre_vector[c(1:4)]
Nombre_vector[c(1,3,7)]
Nombre_vector[-c(1,3,7)]
1.2.1.3.2 Por vectores lógicos
Nombre_vector[c(LOGICO)]

1.2.1.4 Modificar, anexar, remover elementos de un vector

  1. Modificar elementos de un vector
vector <- c(1L:5L)
vector[5] <- 6L
vector
## [1] 1 2 3 4 6
vector1 <- c(1L:5L)
vector1[c(3:5)] <- 6L
vector1
## [1] 1 2 6 6 6
vector2 <- c(1L:5L)
vector2[c(TRUE, FALSE, FALSE, FALSE, TRUE)] <- 6L
vector2
## [1] 6 2 3 4 6
  1. Adicionar elementos a un vector
vector1 <- c(1:3)
vector2 <- c(4:5)
# nuevo vector
vector3_1 <-  c(vector1, vector2)
vector3_1
## [1] 1 2 3 4 5
# mismo vector
vector1[c(4:5)] <- c(4:5)
vector1
## [1] 1 2 3 4 5
  1. Remover elementos de un vector

Indexando negativamente (OJO, para data.frames y listas se puede usar la opción NULL)

# Ejemplo1
vector <- c(1:10)
vector <- vector[-c(6:10)]
vector
## [1] 1 2 3 4 5
# Ejemplo2
vector <- c(1:10)
vector <- vector[c(-6, -7, -8, -9, -10)]
vector
## [1] 1 2 3 4 5
# Ejemplo3
vector <- c(1:10)
vector <- vector[vector<6]
vector
## [1] 1 2 3 4 5

1.2.1.5 Operaciones aritméticas con vectores

  1. Adición, multiplicación y división

Ejemplo suma con vectores

v1 <- c(1:5)
v2 <- c(5:1)
v3 <- c(1:5)
v4 <- c(5:1)
suma <- v1 + v2 + v3 + v4
promedio <- suma/3
promedio
## [1] 4 4 4 4 4

Aritmética con vectores de diferentes longitudes

Reciclaje en operaciones con vectores

v2 <- c(87, 81)
suma <- v1 + v2
suma
## [1] 179 171 171 176 164 173 172
